# Desafio_HTML_DIO

## Sobre o Projeto

Este projeto tem como objetivo mostrar algumas das principais tags HTML, fundamentais para a construção de páginas web.

## Tags Estudadas 👩‍💻

- **Títulos:** `<h1>` a `<h6>`
- **Listas:** `<ol>` e `<ul>`
- **Citação:** `<blockquote>`
- **Texto sobrescrito e subscrito:** `<sup>` e `<sub>`
- **Realce de texto:** `<mark>`
