Aprendendo algumas das principais Tags HTML

O que são Tags HTML?

As tags HTML são os elementos que organizam e definem o conteúdo de uma página web, como textos, imagens e links. Elas mostram o que aparece e como aparece no site.

Vamos conversar um pouco sobre elas

H1 a H6

As tags H1 até H6 são usadas para criar títulos na página. A tag H1 é usada para o título principal, o mais importante. Conforme aumentam (H2, H3, H4, H5 e H6), a importância vai diminuindo. Usamos elas para deixar o texto organizado e fácil de entender. Segue o exemplo do H5 e H6

H5 - Título de nível 5
H6 - Título de nível 6

OL e UL

As tags OL e UL servem pra criar listas. OL é lista ordenada, com números que mostram a ordem das coisas. UL é lista não ordenada, só com marcadores, tipo bolinhas. Segue exemplo:

  1. Primeiro item
  2. Segundo item
  • Item um
  • Item dois

Blockquote

A tag blockquote é usada para exibir uma citação, quando você quer destacar uma frase ou trecho no seu texto. Ela aplica um recuo, indicando que se trata de uma citação. Exemplo:

"Com algumas linhas de HTML, você já começa a dar vida a uma página."

Sub e Sup

As tags sub e sup servem para deixar um texto em formato de subscrito (embaixo) ou sobrescrito (em cima).

  • Sub é usado quando você quer que o texto fique abaixo da linha, como em fórmulas químicas. Exemplo:
    • H2O
  • Sup deixa o texto acima da linha, como em números matemáticos. Exemplo:
    • 102

Mark

A tag mark permite destacar um texto, como se você tivesse passado um marcador amarelo por cima daquela parte. Ela chama atenção para um trecho importante no texto. Por exemplo:

Este é um texto normal, mas este trecho está destacado para chamar a atenção.
